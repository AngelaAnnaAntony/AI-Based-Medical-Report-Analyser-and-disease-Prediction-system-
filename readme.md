# AI-Based Medical Report Analyzer and Disease Prediction System

## 📌 Description
This project analyzes medical reports using OCR and predicts possible diseases using rule-based and AI techniques.

## 🚀 Features
- Upload PDF/Image reports
- Extract medical values
- Detect diseases
- Generate PDF report
- WhatsApp sharing
- Doctor recommendation

## 🛠️ Technologies Used
- Python
- Streamlit
- Tesseract OCR
- Pandas

## ▶️ How to Run
```bash
streamlit run app.py
